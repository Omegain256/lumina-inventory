# Agent Rules

1. **Visual Excellence**: Never use standard browser defaults. Always use curated HSL color palettes and modern typography.
2. **Artifact Review**: Every major logic change requires an Implementation Plan and a Walkthrough artifact.
3. **No Placeholders**: If an image is needed, use the Nano Banana 2 tool to generate realistic product mockups.
4. **Local Testing**: Always run `npm run dev` and use the integrated Browser Agent to verify UI changes after every task.
